Tag Central

Tag Central was created in 2005 as part of Godlikenerd.com. I created it after noticing that 43 Things imported data from other services that are tagged with the same thing. This site aggregates this data from a variety of sources. All one needs to do is provide a keyword that the data should be labeled with.

I moved it to its own domain soon thereafter. Peter Parkes adapted Jon Hicks' Del.icio.us Makeover to the site and it was made the default theme thereafter.

In 2019, I retired the site because I wasn't maintaing it properly. I leave this here for posterity.

The following were the sources for Tag Central.

  • Flickr
  • Smugmug
  • Wists
  • BlogMarks
  • 43 Things
  • del.icio.us
  • Connotea
  • Diigo
  • Upcoming.org
  • Consumating
